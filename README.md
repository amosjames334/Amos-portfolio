## 🚀 Quick Start

1. **Clone the repository**
   ```bash
   git clone https://github.com/username/Amos-portfolio.git
   cd Amos-portfolio
   ```

2. **Open in browser**
   ```bash
   # Open index.html in your preferred browser
   open index.html  # macOS
   start index.html # Windows
   ```

3. **For development**
   ```bash
   # Use a local server for better development experience
   python -m http.server 8000  # Python 3
   # or
   npx serve .  # Node.js
   ```

## 🎨 Customization

### **Colors**
Edit CSS custom properties in `css/style.min.css`:
```css
:root {
    --primary-color: #0BCEAF;
    --primary-dark: #078571;
    --primary-light: #62f7df;
    /* ... other variables */
}
```

### **Content**
- Update personal information in `index.html`
- Replace profile image in `img/profile2.jpg`
- Update resume file in `download/`
- Modify social media links in footer

### **Services & Portfolio**
- Edit services in the Services section
- Update portfolio items with your projects
- Customize filtering categories
